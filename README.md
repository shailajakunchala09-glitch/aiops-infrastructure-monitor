# Screenshots

This folder is where dashboard screenshots go for the GitHub README/portfolio
presentation. After running `docker compose up` and registering a couple of
demo servers (so charts have data), capture these views and drop them in
here with the filenames below — the root `README.md` already references
them:

| Filename | Page | What to capture |
|---|---|---|
| `01-overview.png` | Overview | KPI cards + fleet health donut chart + server table |
| `02-performance.png` | Performance | CPU/memory/disk trend charts for a selected server |
| `03-incidents.png` | Incidents | KPI cards + monthly trend chart + incident table |
| `04-logs.png` | Log Analytics | Level distribution + top failing apps + log search table |
| `05-servers.png` | Servers | Registered server list with status pills |
| `06-swagger.png` | API Docs | `/api/docs` Swagger UI showing the endpoint catalog |

Recommended capture size: 1440x900 or larger, PNG format, browser chrome
cropped out.
